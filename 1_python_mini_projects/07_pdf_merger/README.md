# 📄 PDF Merger Project

This script merges multiple PDF files into a single PDF using PyPDF2.
It asks the user how many PDF files to merge, collects file names one by one,
and then writes all collected PDF pages into a single output file.

## ✅ Requirements

- Python 3
- `PyPDF2` library

Install dependency:

```bash
pip install PyPDF2
```

## ▶️ Run

```bash
python pdf_merger.py
```

Make sure the PDF file names you enter are correct and available in the same
folder as this script, or provide full paths.
For more info, visit https://pypdf2.readthedocs.io/en/3.x/
