# 🧮 Menu Calculator

A simple **menu-driven calculator** built in Python.  
This project reinforces the use of **loops, conditionals, exception handling, and functions** — the fundamentals of interactive console programs.

> 🚀 Built with **basic Python structures** to strengthen modular coding, input handling, and control flow.

---

## 📂 Project Structure

```
05_menu_calculator/
├── menu_calculator.py   # Main executable file
└── README.md            # Project documentation
```

---

## 🛠 Requirements

- Python 3.10 or higher (for match-case support)

---

## 🎮 Features

- Menu-driven interface with **5 options**
- Supports four operations:
  - Addition ➕
  - Subtraction ➖
  - Multiplication ✖
  - Division ➗ (with zero-division handling)
- Input validation for both **menu choices** and **numbers**
- Stores and displays the **last calculated result**
- Exit option to gracefully end the program

---

## 🚀 How to Run

1. Clone this repository or download the files.
2. Open a terminal in the project folder.
3. Run the script:

   ```bash
   python menu_calculator.py
   ```

4. Follow the prompts:
   - Choose an operation (1–5)
   - Enter two numbers
   - Repeat until you select Exit (5)

---

## 🧠 Concepts Used

- **Functions** – modular design
- **While loops** – continuous execution
- **Match-case / if-elif** – control flow (Python 3.10+)
- **Try-except blocks** – input validation
- **Error handling** – invalid input, divide by zero
