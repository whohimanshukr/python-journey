# 📧 Email Generator (Strings Project)

A mini Python project to generate professional email ID suggestions using **string operations only**.

This project is perfect for the **Strings topic** because it focuses on cleaning text, splitting names, joining parts, case formatting, and building patterns.

---

## 📂 Project Structure

```
06_email_generator/
├── email_generator.py   # Main script
└── README.md            # Documentation
```

---

## 🎯 Learning Goals

- Practice `.strip()`, `.lower()`, `.split()`, `.startswith()`, `.endswith()`
- Build new strings with concatenation
- Filter characters using loops + `.isalpha()`
- Generate multiple output patterns
- Remove duplicate strings from a list

---

## 🚀 How to Run

```bash
python email_generator.py
```

Then provide:
1. Your full name
2. Domain name (like `gmail.com` or `mycompany.com`)

---

## 🧠 Example

### Input

- Name: `Himanshu Kumar`
- Domain: `www.gmail.com`

### Output

- `himanshu@gmail.com`
- `himanshu.kumar@gmail.com`
- `himanshu_kumar@gmail.com`
- `hkumar@gmail.com`
- `himanshuk@gmail.com`
- `himanshu123@gmail.com`
- `himanshu.kumar2026@gmail.com`
